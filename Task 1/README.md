# NumPy:
## Concepts:
- What NumPy is and why it's faster than plain Python lists
- Creating arrays: np.array(), np.zeros(), np.ones(), np.arange(), np.linspace()
- Array shapes, ndim, dtype, and reshape()
- Indexing and slicing — 1D, 2D, and 3D arrays
- Broadcasting — how NumPy handles shape mismatches
- Universal functions (ufuncs): np.sqrt(), np.exp(), np.log()
- Aggregations: np.sum(), np.mean(), np.std(), np.min(), np.max() with axis parameter
- Array stacking and splitting: np.stack(), np.hstack(), np.vstack(), np.split()
- Random module: np.random.seed(), np.random.rand(), np.random.randint(),
np.random.normal()
Linear algebra in NumPy: np.dot(), np.linalg.inv(), np.linalg.eig()
