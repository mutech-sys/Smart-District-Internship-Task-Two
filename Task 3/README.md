# Matplotlib & Seaborn:
## Concepts:
- What data visualisation is for and when to use which chart
- Matplotlib basics: plt.plot(), plt.bar(), plt.scatter(), plt.hist(), plt.pie()
- Figure and axes: plt.figure(), plt.subplot(), fig, ax = plt.subplots()
- Customisation: titles, axis labels, legends, grid, color, linestyle
- Saving figures: plt.savefig()
- Seaborn overview — built on Matplotlib, works with DataFrames
- sns.histplot(), sns.boxplot(), sns.violinplot() — distribution plots
- sns.scatterplot(), sns.lineplot() — relational plots
- sns.heatmap() — correlation matrices and confusion matrices
- sns.pairplot() — multi-variable relationship overview
- sns.barplot(), sns.countplot() — categorical plots
- Setting themes: sns.set_theme(), sns.set_palette()
